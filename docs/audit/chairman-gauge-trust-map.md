## The pattern, in one paragraph

The console's *survivability* gauges (north star, runway, screensaver) were built to the NO-DATA-over-fabricated-zero standard and can be trusted. Almost everything else fails toward **confident wrongness**: queries destructured without error checks render zeros ("0 pending", "Score: 0", "0/0 0%"), missing RLS policies render empty pipelines (stage executions), failures render as their own all-clear ("All clear!", "all clear", vanished rows, cleared badges), defaults render as measurements ("50", "0% complete"), and one latency metric renders as strategic strength. The decision-weighted fixes, in order: A2 (one-word select fix restores decision evidence), A3/A4 (never default a blocking signal to calm), A1 (one count source), B1/B2 (RLS policies for chairman-readable ops truth).

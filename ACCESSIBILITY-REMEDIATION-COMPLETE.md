## Maintenance Guidelines

### For New Components
Always include labels when creating form controls:

```jsx
// ‚úÖ GOOD - Explicit label association
<label htmlFor="control-id">Label Text</label>
<input id="control-id" />

// ‚úÖ GOOD - ARIA label
<input aria-label="Descriptive purpose" />

// ‚ùå BAD - No label
```
